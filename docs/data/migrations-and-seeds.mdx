---
title: 'Migrations & Seeds'
description: 'Create, run, and manage database migrations and seeds with the Flowra CLI.'
---

Flowra wraps Knex migrations and seeds with CLI commands that read `orm.cli.config.js`.

## Create a migration

```bash
flowra db:migrate:make create_users_table
```

By default, migrations are stored in `app/Database/Migrations`.

## Run migrations

```bash
flowra db:migrate:latest
```

## Roll back

```bash
flowra db:rollback
flowra db:rollback --all
```

## Migration status

```bash
flowra db:status
```

## Seeds

Create a seed:

```bash
flowra db:seed:make seed_users
```

Run seeds:

```bash
flowra db:seed:run
```

:::note Connection aliases
